# 🌶️ Termina

Simple terminal I/O transmitter to your browser.

### How its works

You can send terminal commands from your browser, even to another device!

### Usage

#### Install packages

```bash
pip install -r requirements.txt
```

#### Run!

```bash
python termina.py
```
